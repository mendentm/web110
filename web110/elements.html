HTML ELEMENTS

Element Description Block, Inline or N/A Container or Empty
<!--...--> On a browser, HTML comments are invisible unless you view its source. Often to give credit or provide instructions. N/A container
!DOCTYPE It's an instruction to the browser about what version the HTML is written in, not an HTML tag. N/A empty
<a> This tag defines a hyperlink, when clicked it links to another page. inline container
<article> This tag is about specific content that is self-contained. block container
<aside> This tag is about content aside from content its placed in. block container
<body> This tag has all the text, images, hyperlinks, tables, lists, etc. block container
<br> Its an empty tag that inserts a single line break. inline empty
<dd> This tag describes a term or name in a description list. block container
<div> This particular tag is used frequently as a container for other HTML elements to style them with CSS or to preform tasks with JavaScript. block container
<dl> This tag is used for a description list, to describe it. block container
<dt> Used in conjunction with dl tag, this defines a term or name in a description list. block container
<em> This tag renders the text as emphasized. inline container
<fieldset> Related elements are grouped with this tag, in a form. block container
<figcaption> This tag is used to define a caption for the figure element. block container
<figure> This tag specifies code, diagrams, photos, etc. block container
<footer> It's an element that typically contains related documents, authorship information, copyright information etc. block container
<form> Its a tag used to create an HTML form for input by the user. block container
<h1 to h6> Its a tag used to by number of importance, define HTML headings. block container
<head> Its an element that can include tags like style, base, meta, script, etc. block container
<header> Its an element that typically contains a logo or icon, authorship information, or heading elements. block container
<html> This tag lets the browser know the document is HTML. block container
<iframe> This tag specifies an inline frame. inline or block container
<img> In an HTML page, this tag defines an image. inline container
<input> Used inside a form element to allow users to input data. inline container
<label> Its an element that toggles control for mouse users. inline container
<li> Its a tag that defines a list item. block container
<link> Contains attributes only, its an empty element. inline container
<meta> Usually specifies keywords, last modified, page description, and other metadata. inline container
<nav> A set of a major block of navigation links. block container
<ol> Either a numerical or alphabetical list that this tag defines. block container
<p> This tag defines a paragraph. block container
<script> This tag defines JavaScript block container
<section> This tag defines sections in a document like headers, footers etc. block container
<strong> This tag defines important text. inline container
<style> This tag defines an HTML documents style information. block container
<table> This tag is a simple table which has two columns and two rows. block container
<td> HTML tables have two kinds of cells, this tag defines a standard cell. block container
<th> HTML tables have two kinds of cells, this tag defines a header cell. block container
<title> A title tag is required in all HTML documents and it defines the title of the document. block container
<tr> This tag defines a row in an HTML table. block container
<ul> This tag defines a bulleted or unordered list. block container
